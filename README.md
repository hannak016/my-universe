# my-universe:
a solar system implementation using ** three.js ** and ** WebGL **

## let's go
* Be sure to have [NodeJS](https://nodejs.org/en/) and [NPM](https://www.npmjs.com/) installed.
* `git pull` to copy the repository
* `npm i` to install dependencies
* `npm start` to run.


## points:
* transformations
* scene graph
* geometry
* texture mapping
* local illumination
* shader programming
